# Bank Account Management System (Java)

This project is a simple Java console application that simulates basic bank account operations. Users can create an account, deposit and withdraw money, check their balance, and view account details through a menu-driven interface.

## Features
- Create a bank account  
- Deposit funds  
- Withdraw funds with validation  
- Check account balance  
- View account details  
- Interactive console menu

## Technologies Used
- Java  
- Object-Oriented Programming (OOP)

## How to Run
```bash
javac mainFile.java
java mainFile
